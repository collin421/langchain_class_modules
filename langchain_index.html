이 커리큘럼의 구성

각 파트는 세 가지 자료로 이뤄짐 — 개념을 다지는 이론 문서, 직접 만져보는 인터랙티브 웹 강의, Gemini로 돌려보는 실습 노트북임. 이론으로 읽고, 웹으로 체감하고, 노트북으로 손에 익히는 흐름임.

📄 이론 .md 🖥️ 웹 강의 .html (섹션 5 · 위젯 5) 📓 실습 .ipynb (Gemini)
🟢 초급 — 기본기Part 0~5 · 정해진 변환(체인)
🟡 중급 — 에이전트Part 6~11 · 자유 판단
🔴 고급 — 제어와 운영Part 12~18 · 흐름 제어·측정·운영
"올바른 도구를, 올바른 자리에, 필요한 만큼만."

자료 사용 안내

  • 학습 순서 — Part 0부터 순서대로. 각 파트는 앞 파트를 딛고 쌓이며, 고급은 초·중급 개념을 계층적으로 결합함.
  • 파트마다 — 이론(.md)으로 개념 → 웹(.html)으로 인터랙티브 체감 → 실습(.ipynb)으로 Gemini 실행. 노트북엔 미니 과제와 정답이 포함됨.
  • 실습 환경 — Gemini API 키 1개면 전 과정 실행 가능. 무료 티어(Flash)로 충분함. LangSmith(Part 16)만 별도 계정이 필요하나, 키 없이도 로컬 미니 평가로 원리를 체험함.
  • API 최신성 — 모든 코드는 LangChain·LangGraph 1.0 기준. 라이브러리는 빠르게 진화하므로 공식 문서(docs.langchain.com)와 함께 확인하는 습관을 권함.
